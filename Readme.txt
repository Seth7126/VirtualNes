
-デバッグ用ウインドウが欲しいとお嘆きのあなたへ(笑)-

========
= 概要 =
========

  Windowsでソフトウェアを開発していて，デバッグ用にウインドウを開いてそこに出力
  したいと思った事はありませんか？

  そんな願いを叶えられるかも知れないソフトウェアです。

  但し，Visual C++でしか試してません^^;

  Win32専用です。98/2Kでテストしました。

================
= セッティング =
================

  同梱のDebugOut.h,DebugOut.cppをあなたのプロジェクトに追加します。

  そのままだと，_DEBUGが定義されていないと出力されないので，リリースビルドの場合
  にも出力したい場合は，_DEBUGOUTを定義しましょう。(プロジェクトオプションで指定
  した方が楽です)

==========
= 使い方 =
==========

  出力したい場合は，DebugOut.hをインクルードし，DEBUGOUTマクロを使います。printf
  と同じ構文が使えるはずです。

  出力を見たい場合は，自分のプログラムを起動する前に，同梱のDebugWindow.exeを起
  動しておきましょう。DebugWindow.exeが起動されていない場合はVisual C++のデバッ
  グウインドウに出力します。

  デバッグウインドウが別プロセスなので，マルチスレッドから呼び出しても大丈夫だと
  思います。

2001/12/13 追加

  出力ウインドウの内容をクリアしたい場合は，DEBUGCLRマクロを使います。
  単に，

  DEBUGCLR();

  と記述すればその時点でクリアされます。なお，Visual C++のデバッグウインドウでは
  クリアされません。

==========
= その他 =
==========

  余りに長い文字列は出力出来ません。(終端のNULLを含め約1000バイトまでです)

  デバッグウインドウ内は256KBのバッファを取っています。スクロールしなくなったら
  クリアのメニューをクリックして下さい。

  フォントはMS ゴシックを使用しています。デフォルトでインストールされていると思
  うので多分問題無いかと。
